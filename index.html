SVG use as a Font Icon Steps

  1. Create a svg file as a icon.svg. You can create the file with any name, but file farmatt as .svg is compulsory.
    Save the file in your project's folder, ex. assets/icon.svg | images/icon.svg | css/icon.svg
  2. In icon.svg file create <svg> then add in the svg (icon) as a <symbol><path id="home-icon"/></symbol>
  3. Place as a icon in your file like index.html icon code will be <svg><use xlink:href="assets/icons.svg#home"></use></svg>. Make sure your file path is correct.
